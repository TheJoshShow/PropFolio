# PropFolio (Swift / SwiftUI)

This directory is the **legacy native iOS** codebase (SwiftUI, pre–Expo migration). It is **not** the current MVP shipping target.

- **MVP app:** **`expo-app/`** (Expo + React Native, iOS/Android/Web).
- **Formulas / scoring:** Ported or duplicated in **`expo-app/src/lib/`** for the cross-platform app; treat Swift engine code here as **reference** unless you explicitly revive this target.

For run instructions if you still build this target in Xcode, see root **`docs/DEVELOPMENT.md`** and **`docs/SETUP-BACKEND-CONFIG.md`** (Swift-specific Supabase wiring).
