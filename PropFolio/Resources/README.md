# Resources

**What belongs here:** Asset catalogs, localizations, and other non-code resources.

- **Assets.xcassets:** App icon, launch screen, images, colors (if using asset colors).
- **Localizable.strings:** All user-facing strings for localization.
- **Info.plist:** App config (or use target settings); Supabase URL/key can be in xcconfig and not committed.

Do not commit API keys; use build configuration or environment.
