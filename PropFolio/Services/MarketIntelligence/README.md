# Services / MarketIntelligence

**What belongs here:** Market-level signals and explanations.

- **Adapters:** Census, BLS, Realtor.com, etc. (or backend that aggregates these).
- **Signals:** Population growth, migration, income, permits, supply/demand; each with source, date range, confidence.
- **Aggregators:** Market health score, category scores.
- **Predictors:** Future value / appreciation scenarios (rule-based, with disclaimer).
- **Explainers:** Rule-based plain-English text per signal (no AI for numbers).

All signals must have provenance (source, date range, confidence).
