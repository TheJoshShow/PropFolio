# Components

**What belongs here:** Reusable SwiftUI views and modifiers used across screens.

- **Buttons:** Primary, secondary, destructive; use DesignSystem.
- **Cards:** Property card, metric card, confidence card.
- **Inputs:** Text field, slider, segmented control (styled).
- **Feedback:** Loading indicator, empty state view, error banner, retry button.
- **Lists:** Row styles for portfolio, property summary.

Keep components presentational; pass data in. Support Dynamic Type and VoiceOver. Minimum 44pt touch targets.
