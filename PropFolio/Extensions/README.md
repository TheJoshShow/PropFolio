# Extensions

**What belongs here:** Swift extensions and small pure helpers used across the app (no business logic).

- **Extensions:** `Date+Format`, `Decimal+Currency`, `String+Validation`, `View+Modifiers`.
- **Helpers:** Formatters (currency, percent), validation helpers.
- **Constants:** App-wide constants (e.g. max input lengths) that don’t belong in DesignSystem.

This folder was named **Extensions** (not Utilities) to match the roadmap and Swift convention. No business logic; only pure helpers and extensions.
