# Test Helpers

**What belongs here:** Shared test utilities.

- **Fixtures:** Sample UnderwritingInputs, NormalizedProperty, MarketSignal for tests.
- **Assertions:** Custom XCTAssert helpers (e.g. assertDecimalEqual with scale).
- **Async helpers:** Wait for async in tests (XCTestExpectation or async/await).

No production code; only used by tests.
