# App Store Release — PropFolio

This folder contains materials for **App Store submission**, TestFlight, and release.

| Document | Purpose |
|----------|---------|
| **app_store_metadata.md** | App name, subtitle, keywords, description, What's New, URL placeholders for Connect. |
| **review_notes.md** | Text for App Review (purpose, test account, subscription/import/API notes). |
| **screenshot_plan.md** | Screens to capture, order, captions, free vs premium. |
| **testflight_plan.md** | Internal/external beta checklists and feedback instructions. |
| **release_checklist.md** | Version, build, signing, legal links, subscription, crash-free. |
| **submission_audit.md** | Audit: icons, launch screen, permissions, privacy, links, placeholder copy. |

**Before first submission:** Run through `submission_audit.md` and fix any missing assets (see `expo-app/assets/images/README.md`). Use `release_checklist.md` for each version you submit.
